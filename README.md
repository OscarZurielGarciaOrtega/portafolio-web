##  Descripción del Proyecto
Este proyecto consiste en un **Portafolio Web Personal Profesional** estático y completamente responsivo. Fue adaptado a partir de una plantilla base para estructurar de manera estética mi perfil técnico, habilidades y proyectos de nivel ingeniería.

* **Framework CSS utilizado:** Bootstrap 5
* **Plantilla base:** SnapFolio 
* **Enlace de descarga de la plantilla original:** [Descargar SnapFolio en BootstrapMade](https://bootstrapmade.com/snapfolio-bootstrap-portfolio-template/)

###  Secciones del Portafolio
1. **Inicio / Hero:** Pantalla de bienvenida con efecto de tipografía dinámica, enlaces directos a mis redes profesionales (GitHub) y una fotografía formal optimizada para CV/LinkedIn.
2. **Sobre Mí (Conóceme):** Resumen profesional sobre mi formación académica en el ITO, especialización técnica en desarrollo web, nivel de experiencia e idiomas.
3. **Habilidades Técnicas (Skills):** Barras de progreso interactivas con tooltips detallados que describen mi dominio técnico dividido en:
   * **Desarrollo Front-end:** HTML5/CSS3 moderno, JavaScript y React.
   * **Desarrollo Back-end:** Java (POO), Firebase y bases de datos relacionales y NoSQL (PostgreSQL y MongoDB).
4. **Proyectos:** Galería visual que destaca mis tres desarrollos reales e independientes:
   * **MotoTec:** Sistema web de inventario y gestión para un taller de servicio de motocicletas.
   * **Xquenda:** Arquitectura y administración de bases de datos relacionales.
   * **Sistema Textil:** Software a medida para la administración, venta y renta de ropa tradicional oaxaqueña.
5. **Contacto:** Formulario estático maquetado y datos de contacto directo (ubicación geográfica en Oaxaca, teléfono y correo electrónico).

---

##  Proceso de Creación 
1. **Descarga y Reestructuración:** Se obtuvo la plantilla original en formato ZIP y se realizó una reestructuración profunda de los directorios. Se migraron los recursos esenciales desde la carpeta nativa `assets/` hacia la estructura estricta solicitada por la rúbrica (`css/`, `js/`, `img/`).
2. **Inyección de Dependencias (Vendor):** Se mantuvo la carpeta `vendor/` en la raíz del proyecto para asegurar el funcionamiento de Bootstrap Icons y las librerías de animación. Se actualizaron las rutas absolutas y relativas en el archivo `index.html` para que apuntaran correctamente a las nuevas carpetas.
3. **Depuración de Código (Modificaciones):** Se eliminaron bloques de código genérico que la plantilla traía por defecto (secciones de servicios de agencias, testimonios falsos y menús desplegables sin uso). Esto se hizo con la finalidad de mantener una interfaz limpia, honesta y adaptada a un perfil real de estudiante de ingeniería.
4. **Personalización Técnica:** Se tradujo el 100% de la interfaz al español. Se recalibraron los scripts de las barras de progreso para que coincidieran visualmente con mis niveles reales de dominio técnico y se redactó el contenido basándome en mis proyectos.
5. **Optimización de Recursos Visuales:** Se integró una fotografía formal con fondo neutro, cumpliendo con los estándares de un perfil profesional, y se configuró el pie de página (*footer*) con los créditos correspondientes.
